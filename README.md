AI-based diet recommendation system using Knowledge-Based System (KBS). 
Includes calorie tracking, food recommendation, and exercise tracking.

Fun Diet AI adalah aplikasi berbasis Knowledge-Based System (KBS) untuk memberikan rekomendasi diet dan olahraga berdasarkan data pengguna.
